Cryptographic Specifications

1. Symmetric Architecture

Hides uses the AES-GCM (Galois/Counter Mode) cipher with a 256-bit key size. This symmetric cipher provides both confidentiality and authenticated data integrity. The encryption process uses the client-side SubtleCrypto API, executing isolated in-browser threads.

2. IV (Initialization Vector) Isolation

A fresh, cryptographically secure 12-byte (96-bit) Initialization Vector (IV) is generated via crypto.getRandomValues() for every single encryption operation. Under no circumstances is an IV ever reused with the same key, preventing ciphertext correlation or XOR-based cipher compromises.

3. Share Token Encoding

The generated base64 token is a packed binary sequence constructed as follows: 

[1-byte Key Length] [32-byte Raw Key] [12-byte IV] [Ciphertext Byte Stream]

Since the entire token is processed locally, the keys never traverse the network. Your data remains strictly zero-knowledge.
